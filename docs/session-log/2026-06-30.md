구현 방식: 새 탭으로 `http://localhost:{할당된포트}` 오픈 (iframe 방식보다 단순, 추후 업그레이드 가능)

### 버그 수정 + 기능 완성

**quest_attempt 기록 누락 수정**

- `gradeQuest` API 호출 시 `questSetId`, `sessionId` 누락 → DB FK 제약으로 INSERT 실패하던 버그 수정
- `crypto.randomUUID()` → HTTP 환경에서 보안 컨텍스트 제한으로 동작 안 함 → `Date.now() + Math.random()` 방식으로 대체
- `catch {}` → `catch (e) { console.error(...) }` 로 변경하여 에러 가시성 확보

**elapsed_sec 구현**

- `QuestPanel.tsx` — `useRef(0)` + `useEffect`로 퀘스트 진입 시각 기록
- 채점 시 경과 시간(초) 계산해서 백엔드 전달
- `useRef` 초기값에 `Date.now()` 직접 호출 시 React 린터 에러 → `useEffect`로 우회

**랭킹 리디자인**

- 리더보드 → 랭킹으로 명칭 변경 (SideNav, 페이지 타이틀, 테이블 헤더)
- `getLeaderboard` 쿼리 개선: `유저 × 세트` 나열 → 유저별 전체 달성률 집계 + 세트 상세 중첩
- 달성률 순 정렬, 유저 행 클릭 시 세트별 상세 아코디언 펼치기

**SideNav 사용자 정보 표시**

- 로그아웃 버튼 위에 로그인한 사용자 이름 + 이메일 표시
- JWT payload에 `name` 누락 → `auth.ts`에서 토큰 발급 시 `name` 포함하도록 수정
